\cvsect{Who Am I?}

\begin{minipage}[t]{1.0\textwidth}
    \vspace{-\baselineskip}
    I’m a software developer based in Milan, originally from Treviso (Italy). I’ve been coding for more than half my life, and it’s something I genuinely enjoy. I’m comfortable building, maintaining, and improving software in a bunch of languages like C, C#, Python, Elixir, Rust, and more — lately, I’ve been doing a lot in Elixir. I’m especially interested in working more with Haskell, Rust, Erlang, OCaml, and Scheme. I really believe in the importance of open source and like being part of that world. These days, I’m mostly into functional programming, algorithms, compilers, interpreters, DSLs, VMs, and autonomous robots. I enjoy designing and deploying backend services that are scalable, low-latency, and highly available. I'm also into building infrastructure, CI/CD pipelines, and using different testing strategies, including A/B testing. I'm always trying to stay on top of best practices and enjoy running software in Linux environments. I care about solid engineering principles like continuous delivery, defensive programming, and automated testing. I've got experience building distributed systems, and there’s something really satisfying about knowing your code is being used by millions of people. I’ve got a strong grasp of data structures and algorithms, and I’m all about using them to build practical, real-world solutions.
\end{minipage}
